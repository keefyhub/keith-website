Projects

Dirtees - Crude T-Shirt Design Store

Dirtess is a small company based in Gosport, Hampshire that create crude, satirical and parody t-shirts.

The website is built on a CMS called Sellerdeck (formally known as Actinic), this was chosen to manage the large amount of products, deal with the checkout process and keep it secure. Using Sellerdeck has made it slightly more challenging due to the way it complies the data, it uses PERL for the backend which complies the static pages using templates.

Features

  • Content Management System
  • Integrated Blog System
  • Custom Light Box (jQuery)
  • jQuery running total when adding quantity
  • Custom CSS and Layout integrated with Sellerdeck
  • Managed Hosting and email
  • Support for older browsers
  • Google Analytics

Ela Rae Music - Classical Wedding Musicians

Ela Rae Music is a London based collection of classical musicians who specialise in high-end weddings,functions and events.

The website is built from scratch using PHP to compile the pages, I built a contact form with a "honey trap" capture to avoid the need of a CAPTCHA box. The website is built on a CSS grid system (LANES), which allows it to be fully responsive across all devices. I also took advantage of HTML5 properties for better user experience (inputs, structure etc).

Features

  • Bespoke design, CSS grid system
  • Fully Responsive Design
  • PHP generated pages and content
  • PHP contact form - with honey trap
  • CSS animations, jQuery functions
  • BX-Slider for rotating carousel
  • Managed Hosting and email
  • Support for older browsers
  • Google Analytics

Stitch Republic - Badges Website

Stitch Republic supplies embroidered, woven, printed and rubber badges to clubs, teams schools and special events.

The website is built from scratch using PHP to compile the pages, features a PHP contact form and PHP order form with image upload. Both PHP forms use the "honey trap" technique for SPAM filtering. The website is built on a CSS grid system (LANES), which allows it to be fully responsive across all devices. The responsive menu has 3 break points so small, medium and larger screens.

Features

  • Bespoke design, CSS grid system
  • Fully Responsive Design
  • Custom Light Box (jQuery)
  • PHP generated pages and content
  • PHP contact & order form - with honey trap
  • CSS animations, jQuery functions
  • Managed Hosting and email
  • Support for older browsers
  • Google Analytics

Embroidery Mill (redesign) - embroidery designs

Embroidery Mill is the UKs largest embroidery stock design library.

Originally built (by me) a few years back, the standard template provided by sellerdeck is a bit dated. This redesign features a new layout, colour co-ordinated headers, buttons and menu, a new contact form and newsletter sign-up and some custom javascript functions. Embroidery Mill is also built on Sellerdeck, which manages the backend and data-storage. As there is over 3000 products it needed something robust to manage all the products.

Features

  • Content Management System
  • Pop-Out PDF script
  • Newsletter Sign-Up
  • jQuery scroll-to-top button
  • Custom CSS and Layout integrated with Sellerdeck
  • Managed Hosting and email
  • Support for older browsers
  • Google Analytics
